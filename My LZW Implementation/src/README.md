# Modified LZW Implementation Source Files 

These are the source files for running this `LZW` implementation. 

Compilation: 

    javac MyLZW.java 

    javac LZW.java 

Execution: 

    java LZW - < input.txt   (compress)

    java LZW + < input.txt   (expand)
